## 🚀 Overview

NanoBoard Pro is a developer-focused Android keyboard designed to make coding and productivity easier on mobile devices.

Unlike traditional keyboards, NanoBoard Pro provides quick access to programming keys and shortcuts commonly used by developers.

---

## ✨ Features

✅ Dedicated Ctrl Key

✅ Dedicated Alt Key

✅ Dedicated Shift Key

✅ Tab Key Support

✅ Coding Symbols

✅ Fast & Lightweight

✅ Dark Developer Theme

✅ Mobile Coding Friendly

✅ Clean User Interface

---

## 🛠️ Built With

* Java
* Android SDK
* XML
* AIDE (Android IDE)

---

## 📦 Installation

1. Download the latest APK.
2. Install on your Android device.
3. Enable NanoBoard Pro from Keyboard Settings.
4. Start coding efficiently.

---

## 🔐 Privacy

NanoBoard Pro is designed with user privacy in mind.

* No unnecessary data collection
* No keystroke logging
* No personal information sharing
